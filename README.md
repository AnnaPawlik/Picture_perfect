# Picture_perfect

Simple website layout

## Technology:
- HTML5
- Sass

## Getting started:

Visit: https://annapawlik.github.io/Picture_perfect/

OR 

Clone repository
```
git clone git@github.com:AnnaPawlik/Picture_perfect.git
```

Install dependencies
```
npm install
```

Running development
```
// in main folder
npm run watch
```
